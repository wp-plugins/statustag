== Installation ==

1. Upload `statustag.php` & `statustag_icon.png` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. This will replace `<?php bloginfo('description'); ?>` with your latest twitter post.
